CryptoPro — Config Supabase

Connectez votre projet Supabase pour activer la synchronisation cloud. Cette configuration est unique.

01Créez un projet gratuit sur supabase.com
02Dans Project Settings → API, copiez l'URL et la clé anon public
03Dans l'éditeur SQL Supabase, exécutez ce script :
-- Tables create table public.positions ( id uuid primary key default gen_random_uuid(), user_id uuid references auth.users(id) on delete cascade not null, coin_id text not null, amount numeric not null, buy_price numeric not null, buy_currency text not null default 'usd', created_at timestamptz default now() ); create table public.alerts ( id uuid primary key default gen_random_uuid(), user_id uuid references auth.users(id) on delete cascade not null, coin text not null, condition text not null, price numeric not null, currency text not null default 'usd', active boolean default true, triggered boolean default false, created_at timestamptz default now() ); -- Sécurité (RLS) alter table public.positions enable row level security; alter table public.alerts enable row level security; create policy "own positions" on public.positions for all using (auth.uid() = user_id); create policy "own alerts" on public.alerts for all using (auth.uid() = user_id);

Stratégies d'investissement

DCA — Dollar Cost AveragingRecommandé

Investir une somme fixe à intervalles réguliers, quelle que soit le cours. Réduit l'impact de la volatilité sur le long terme.

  • Choisir un montant hebdomadaire ou mensuel (ex : 50 €)
  • Sélectionner 1 à 3 actifs de premier rang (BTC, ETH...)
  • Configurer un achat automatique récurrent sur votre exchange
  • Ne pas vendre lors des corrections — tenir le cap
Risque faibleHorizon : 2 ans+
Rééquilibrage de portefeuilleIntermédiaire

Maintenir des allocations cibles en vendant les actifs surpondérés et en achetant les sous-pondérés.

  • Définir vos allocations cibles (ex : 60% BTC, 30% ETH, 10% Alts)
  • Rééquilibrer tous les trimestres ou si un actif dévie de ±20%
  • Prendre des bénéfices progressivement sur les actifs surperformants
  • Conserver un minimum de 20% en stablecoins pour saisir les dips
Risque modéréHorizon : 1–3 ans
Pyramiding — Renforcement en baisseAvancé

Acheter des volumes croissants lors des baisses successives pour abaisser son prix moyen d'acquisition.

  • Diviser le capital prévu en 4 à 6 tranches
  • Déployer la première tranche à cours actuel
  • Chaque palier : acheter 2× la tranche précédente si -15%
  • Définir un stop-loss absolu pour protéger le capital
Risque élevéRéservé aux experts
